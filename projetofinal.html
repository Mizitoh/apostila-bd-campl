Projeto Final Primeira Turma

Banco de dados Hospital

Contexto:
Você foi contratado para modelar um pequeno sistema de cadastro de consultas para um hospital. Seu banco vai ser testado no programa criado pelo professor, se funcionar, além de aprovado, recebe o prêmio combinado.

  1. Crie o banco de dados com seu nome.
  2. Criar as tabelas.
  3. Fornecer ao menos 5 comandos INSERT para cada tabela.
  4. Fornecer as seguintes análises de dados (SELECT):
    • Quantidade de consultas por médico.
    • Quantidade de pacientes atendidos por especialidade.
    • Faixa etária dos pacientes atendidos.
    • Número de pacientes com convênio vs sem convênio.
    • Médicos mais generalistas (com mais especialidades).
    • Médicos que não possuem consultas registradas.

Exemplo:


-- Crie o banco de dados com seu nome
CREATE DATABASE nome_aluno;

-- Tabela 
CREATE TABLE ...

--Inserts
INSERT INTO ...

--selects
SELECT ...

>Execute os comandos no parquinho e avise o professor.


Prazo pra entrega é até o ultimo dia de aula
