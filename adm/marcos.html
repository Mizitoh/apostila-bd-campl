3 - Localize as tabelas que tem a coluna id_loja, usando o comando de metadados abaixo: 

SELECT
	*
FROM 
	information_schema.columns c
WHERE
	c.table_catalog = 'nome_do_banco_de_dados' -- geralmente vr
	AND table_schema LIKE 'nome_do_esquema%' -- existem vários esquemas, 
	AND TABLE_NAME LIKE '%%'                 -- como public, implantacao, etc...
	AND column_name LIKE '%id_loja%'
ORDER BY
	1,2,3,4
